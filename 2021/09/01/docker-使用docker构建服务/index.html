docker-使用docker构建服务

发表于2021-09-01|更新于2022-12-09|Docker
|阅读量:

停止运行的容器
docker ps | awk ‘{if(NR!=1)print $1}’ | xargs docker stop

删除已经停止的容器
docker ps -a | awk ‘{if(NR!=1)print $1}’ | xargs docker rm
