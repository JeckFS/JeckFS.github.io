openGauss-管理及开发工具

发表于2022-09-15|更新于2022-12-09|数据库
|阅读量:

gsql -l sql-name.sql ：直接运行 sql-name.sql 中的语句。
gsql -1 参数：一般与 -l 参数配合使用， -l 指定的 sql 文件执行时所有文件内的语句放在一个事务中执行，若出错则整个文件的 sql 都回滚。
gsql -L query.log ：将本次会话中所有的执行语句的结果存放在 query.log 中
gsql -A ：输出结果不进行对齐，而是逐个字符输出。
gsql -t：仅显示数据行，不显示列名
gsql -L：指定数据行之间列分隔符
gsql -H：查询结果以 HTML 格式进行输出

gsql -x：同 mysql \G

\copy：将查询输出导出到文件中文件中。或者将 copy 出的结果导入到另一个表中。
\i：执行指定的 sql 文件中的脚本
\d：显示所有的关系表
\d tablename：显示 tablename 的详细信息

\a：切换对齐方式
\x：切换输出是按行还是按列
\t：切换是否显示列名
