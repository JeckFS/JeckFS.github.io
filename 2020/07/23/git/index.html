git

发表于2020-07-23|更新于2022-12-09|git
|阅读量:

https://www.bilibili.com/video/BV1fK4y1b7XL?p=2

Git是分布式版本控制工具，没有中央服务器，每个人存有完整的版本库。对于在不同机器上的修改，只推送各自的修改部分到远程仓库，然后从远程仓库pull。

常用命令

  1. git status
    显示文件的更改状态。

  2. git diff

- 红色部分是修改前的文本。
+ 上方紧挨着的一行修改后的部分。

  1. git add
  2. git commit -m “description”
    命令3,4用于提交到本地仓库
  3. git log 显示提交日志
  4. git log —pretty=oneline 显示简略的提交日志

HEAD表示当前版本，即最新提交的版本
HEAD^：上一个提交版本
HEAD^^：上上个提交版本
HEAD~100：上100个提交版本

  1. git reset —hard HEAD^ 版本回退到上一个版本
  2. git reset —hard 7b258 回退到版本id以7b258开头的版本
  3. git reflog 查看命令历史，可以找到以前所有的版本
